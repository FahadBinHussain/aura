# Research Plan: ArtStation Integration

Main question: What public ArtStation endpoints and response shapes can Aura use to browse artwork images in the WinUI app?

Subtopics:
1. Current public ArtStation project list/search endpoints and supported parameters.
2. Current project/detail JSON shape and image fields suitable for thumbnails/full images.
3. Practical integration constraints such as rate limits, blocking, user agents, and fallback behavior.

Synthesis: Validate endpoints with live requests, then implement the WinUI integration using the existing platform/page patterns.
